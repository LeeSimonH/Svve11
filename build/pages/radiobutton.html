Radio Button

  • Source: https://github.com/Svve11/
  • WAI-ARIA: https://w3c.github.io/aria-practices/#radiobutton

A radio button is a clickable widget that is used to gather user input or responses, and they are typically used within a group of radio buttons. Only one of the radio buttons can be checked at a time.

Installation

Installation

Import the component in the script section of your Svelte file:


              import ❴ RadioButton ❵ from 'svve11'
Usage

Usage

Creating a Radio Button

A radio button instance can be created by placing the code below in the body of your Svelte file. 


              <RadioButton {radioButtonLabel} {id} {checked} {radioButtonStyle} {radioButtonLabelStyle} />

Each attribute should be defined as follows:

  • id (string): sets the id attribute of the radio button component.
  • name (string): sets the name attribute of the radio component. This should be the same string for each radio button within a group.
  • radioButtonLabel (string): sets the aria-label attribute of the radio button component.
  • checked (boolean): sets the initial state of the radio button, where true will render a pre-checked button and false will render a non-checked button.
  • radioButtonStyle (string): sets the styling for the radio button
  • radioButtonLabelStyle (string): sets the styling for the radio button label text.

Example Radio Button Code:


		<RadioButton
		  name="food"
		  value={1}
		  radioButtonLabel="Pizza"
		  id="radioButtonOne"
		  checked={false}
		  radioButtonStyle="height: 1.5em; width: 1.5em;"
		  radioButtonLabelStyle="font-size:1.5em;"
		/>
