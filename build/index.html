Welcome to Svve11

Here at Svve11, we believe that accessible web development is the future. We know that creating and designing accessible web pages can be a challenge, thus we have created a library of Svelte components built with an accessibility first design strategy. These components can be easily imported into your Svelte files and quickly rendered to your web pages.

Installation

Installation

For npm, run the following in your command line

  
          npm install 'svve11'

For yarn, run:


          yarn add 'svve11'
